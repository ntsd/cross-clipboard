# Cross Clipboard

A multi device clipboard sharing over p2p in lan network.

## Technology

- Go
- libp2p
- Multicast DNS (mDNS)
- Fyne

## Installation

- Install NDK 21.3.6528147

- Install Fyne

```shell
go install fyne.io/fyne/v2/cmd/fyne
fyne install
```

- Install Fyne Cross

```shell
go install github.com/fyne-io/fyne-cross
fyne-cross android
```
