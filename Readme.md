# Cross Clipboard

A multi device clipboard sharing over p2p in lan network.

## Technology

- Go
- libp2p
- Multicast DNS (mDNS)
- Fyne

## Installation

- Install NDK >=21.3.6528147

- Install Go mobile

```shell
go install golang.org/x/mobile/cmd/gomobile@latest
```
